# Campus Manager

**Campus Manager** is a university management game where players create, build, and manage their own university. From designing departments and hiring faculty to managing finances and developing facilities, youâ€™ll be in charge of everything that makes a successful academic institution.

## Features
- Create and manage university departments
- Build and upgrade campus facilities
- Hire staff and recruit students
- Track research quality, industry collaboration, and international outreach

## Technologies Used
- HTML, CSS, JavaScript (Frontend)
- GitHub Pages (for hosting)
- LocalStorage (for saving game progress)

## How to Play
1. Start a new game by creating your university.
2. Develop your campus, build facilities, and recruit faculty.
3. Track your university's growth and rankings over time.
4. Aim to become the top university in the world!
